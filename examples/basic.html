VIB34D Engine - Basic Example

This example demonstrates basic usage of the VIB34D Holographic Engine.

External Controls

These buttons control the engine from outside the iframe:

Code Example:

// Access the engine inside the iframe
const frame = document.getElementById('engineFrame');
const engine = frame.contentWindow.engine;

// Set a specific variation
engine.setVariation(15);

// Update parameters
engine.updateParams({
    gridDensity: 20.0,
    morphFactor: 0.8,
    chaos: 0.5
});

// Export current configuration
const config = engine.exportJSON();
